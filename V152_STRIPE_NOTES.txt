Required Cloudflare environment variables:
- STRIPE_SECRET_KEY = your Stripe test or live secret key
- STRIPE_PRICE_ID = price_1THVwSRLL3yuhoRUfdZCX0dz
- BASE_URL = your site origin, for example https://bracketologybuilder.com

Current activation flow:
- user clicks Unlock for $5.99/month
- site creates a Stripe Checkout session
- Stripe sends the user back to /subscription-success.html with the session_id
- the success page calls /api/stripe-complete to mark the signed-in user active

This version does not yet add a Stripe webhook. Unlock finalization happens on the return page after checkout.
